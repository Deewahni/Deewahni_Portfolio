The Durian Fruit

Go to Website

This website shows information about the marvelous fruit, Durian. Some of the many varieties were included in this website. This is created with the use of HTML, CSS, and GSAP for its animation. It was my first time creating a website using GSAP to create animation. It was such a thrill to learn such a plugin. I had hard time manipulating the controls to make it the way I want it done, it sure was tough. But through reseach and watching a bunch of tutorials, I have made it through.
