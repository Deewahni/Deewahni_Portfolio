Employee Information Management System

Go to Website

This system is a collaboration work. It initially stores data ino the server. As the name implies, employee information management is its main purpose. A client requested to have a system where employees upload their information required by the organization. From basic information to things like SSS, SAL-N and whatnots, to name a few. This system is able to upload and store any document files such as word, excel, pdf, and images.
Since it is designed to store individual information, we designed the system to have log-in/log-out setup where every individual has their own account. We made the layout as simple as possible so that the not-so-techy employees will not have hard time operating it. We also created a dashboard for the administrator of this sytem. It's function is to be able to oversee and check all the important files uploaded as well as the information provided.
